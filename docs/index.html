Calendar to Google Calendar

A Chrome extension that extracts calendar events from CapEdge and exports them as .ics files for Google Calendar import.

⬇ Download Extension (.zip)

Features

  • Extracts calendar events automatically from CapEdge dashboard
  • Auto-retry mechanism for slow-loading calendars
  • Filter events by type: Earnings, Financial (10-K, 10-Q), Ownership, Registration, News
  • Downloads standard .ics file compatible with Google Calendar
  • Side panel UI that stays open while you browse
  • Refresh button clears stale data and re-extracts fresh events

Installation (Developer Mode)

  1. 1 Download the extension zip file using the button above
  2. 2 Extract the zip file to a folder on your computer
  3. 3 Open Chrome and go to chrome://extensions
  4. 4 Enable Developer mode using the toggle in the top-right corner
  5. 5 Click Load unpacked and select the extracted folder
  6. 6 The extension icon will appear in your Chrome toolbar

How to Use

  1. 1 Click the extension icon to open the side panel
  2. 2 Navigate to your CapEdge dashboard — events are extracted automatically
  3. 3 Use the checkboxes to filter events by type (Earnings, Financial, etc.)
  4. 4 Click Download .ics File to save the calendar file
  5. 5 Import into Google Calendar: Settings → Import & export → Import
